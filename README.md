# zoomie

Zoomie lets agents store markdown content and get shareable URLs. Upload a report, summary, plan, or any markdown output and get back a live link instantly. No account or API key required.

## Install

```bash
npx skills add zoomie-sh/skill --skill zoomie -g
```

## Docs

Full documentation: **https://zoomie.sh/docs**
